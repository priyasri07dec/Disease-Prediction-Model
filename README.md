# Disease_Prediction_Model

A machine learning based Disease Prediction System built using Python, Scikit-learn and streamlit.
The application predicts disease based on user selected symptoms using multiple classification algorithm.

## Project Overview

This project uses supervised machine learning algorithms to predict disease from symptom data.
Multiple algorithms were trained, evaluated and compared using:
* Training accuracy
* Testing accuracy
* Cross validation
The best performing model was selected and deployed using streamlit.

## Features

* Disease prediction based on symptoms
* Multiple machine learning algorithm comparison
* Cross validation evaluation
* Streamlit web application
* User friendly interface
* Real time disease prediction

## Technologoes Used

* Python
* Pandas
* Numpy
* Scikit-learn
* Streamlit
* Joblib

## Machine Learning alogorithms Used

* Random Forest Classifier
* Logistic Regression
* Support Vector Machine(SVM)
* K-Nearest Neighbours(KNN)
* Naive Bayes
* Decision Tree Classifier

## Dataset Information

* Symptoms as input features
* Disease names as target labels

Each Symptom is represented using binary values
### Value: 0,1
### Meaning : Symptom Absent, Symptom Present

## Project Workfolw

Data Collection -> Data Preprocessing -> Feature Scaling -> Model Training -> Algorithm Comparison -> cross Validation -> Best Model Selection -> Model Saving -> Streamlit Deployment

## Model Evaluation

The models were evaluated using:
* Accuracy Score
* Classification Report
* Cross Validation

## Streamlit Application

The Streamlit application allows users to:
1. Select symptoms from a dropdown list
2. Submit symptoms using the Predict button
3. View the predicted disease instantly
